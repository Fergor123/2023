This folder contains an independent study within which the goal was to figure out how
a peer-to-peer connection would establish and be implemented.
This project utilized Unity game engine and using Unitiy's Netcode for GameObjects.

The game consists of of basic proof of concept mechanics as follows:
-Peer-to-Peer connection on local machine using the Netcode library [Completed]
-Pick-up behavior which is translatable between host and player (E key) [Completed, can be bugged]
-Grappling Hook [Physics translate, visuals partially]
